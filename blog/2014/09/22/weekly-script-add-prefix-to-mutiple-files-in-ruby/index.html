使用的方法

  • ruby add_prefix_files.rb dest_folder prefix pattern
  • dest_folder 必选 操作进行的基础目录，并不一定总是直接父目录
  • prefix 必须 前缀名称 建议结尾以_结束
  • pattern 可选，如不填写为dest_folder的直接子文件（含目录），否则应用提供的pattern匹配
