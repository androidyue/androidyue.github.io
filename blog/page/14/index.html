原始定义

该原则最早由Jerome Saltzer提出。其最原始的表述为 Every program and every privileged user of the system should operate using the least amount of privilege necessary to complete the job.
其中文意思为
系统的每个程序或者用户应该使用完成工作所需的最小权限工作。
